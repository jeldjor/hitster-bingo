Hitster Bingo CSV Spotify Update

Upload deze 3 bestanden naar je bestaande GitHub repository:
- index.html
- style.css
- app.js

Gebruik:
1. Open de website op iPhone.
2. Login met Spotify.
3. Tik Activeer Spotify-speler.
4. Upload je CSV via de bestandenkiezer.
5. Start ronde.

Let op:
Spotify Web Playback SDK vereist Spotify Premium. Op iPhone kan Spotify soms strenger zijn met web-audio; als dit problemen geeft, testen we daarna een fallback die afspeelt op je actieve Spotify-apparaat.
